
# Debug Files

Jogo investigativo em CLI onde cada caso representa uma fase.

O jogador analisa códigos bugados, investiga pistas e tenta descobrir a causa do problema para corrigir o sistema.

Cada caso foi pensado para treinar erros comuns e importantes em Python.

---

# Estrutura do Jogo

```text
MENU
 └── Selecionar caso
      └── Narrativa
           └── Código bugado
                └── Investigar pistas
                     └── Tomar decisão
                          └── Corrigir bug
                               └── Resultado
                                    └── Próximo caso
```

---

# Sistemas do Jogo

* Sistema de vidas
* Sistema de energia
* Narrativa interativa
* Save em JSON
* Progressão por casos

# Casos Planejados

## Caso 01 — Sistema Bancário

### Tema

Cálculo de saldo incorreto.

### Foco Técnico

* Variáveis
* Atribuição
* Operadores
* Cálculos incorretos

---

## Caso 02 — Cadastro Infinito

### Tema

Sistema travando usuários.

### Foco Técnico

* Loops
* Condição de parada
* Loop infinito

---

## Caso 03 — Login Fantasma

### Tema

Usuários válidos não conseguem entrar.

### Foco Técnico

* Condicionais
* Operadores lógicos
* Comparações incorretas

---

## Caso 04 — Inventário Corrompido

### Tema

Itens desaparecendo do sistema.

### Foco Técnico

* Listas
* Mutabilidade
* Remoção e alteração de dados

---

## Caso 05 — O Caso Final

### Tema

Sistema principal comprometido.

### Foco Técnico

* Mistura de bugs
* Debugging completo
* Análise lógica mais complexa
