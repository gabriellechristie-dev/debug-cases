# 🕵️ Debug Cases

Um jogo de investigação de bugs desenvolvido em Python.

Neste projeto, o jogador assume o papel de um investigador responsável por analisar sistemas com falhas, coletar pistas e identificar a causa de diferentes bugs encontrados no código.

Cada caso apresenta uma situação inspirada em problemas comuns de programação e desafia o jogador a descobrir a origem do erro através da análise de código, comportamento observado e pistas fornecidas durante a investigação.

---

## Objetivo

Praticar lógica de programação, depuração de código (debugging), modularização e organização de projetos em Python por meio de uma experiência interativa baseada em investigação de bugs.

---

##  Funcionalidades

* Menu principal
* Campanha com 5 casos de investigação
* Sistema de pistas
* Diagnóstico de bugs
* Repetição automática em caso de erro
* Progressão entre os casos
* Mensagem de conclusão da campanha

---

##  Casos Disponíveis

### Caso 01 — Sistema Bancário

**Conceito:** Variáveis

Investigue um sistema bancário onde depósitos não estão sendo armazenados corretamente.

### Caso 02 — Loop Infinito

**Conceito:** Estruturas de repetição (while)

Descubra por que um sistema de processamento de pedidos nunca encerra sua execução.

### Caso 03 — Login com Falha

**Conceito:** Operadores lógicos

Analise um sistema de autenticação que permite acessos indevidos.

### Caso 04 — Cadastro de Alunos

**Conceito:** Listas

Investigue o desaparecimento de registros após uma atualização de cadastro.

### Caso 05 — Sistema de Boletins

**Conceito:** Funções

Descubra por que boletins não estão sendo gerados mesmo com a funcionalidade implementada.

## ▶️ Como Executar

1. Clone este repositório:


2. Acesse a pasta do projeto:

```bash

```

3. Execute o arquivo principal:

